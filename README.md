# Arizona Deluxe Painting LLC Website

A beginner-friendly static website project built with HTML, CSS, and JavaScript.

## Pages
- `index.html`: landing page and instant quote form
- `contact.html`: message page with call and Instagram links
- `styles.css`: all styling
- `script.js`: quote calculator, image previews, confetti, and mail message logic

## Quote Formula
The estimate is calculated as:

`home square footage × $2`

Example: `1800 sq ft × $2 = $3,600`

## How to customize
1. Replace the placeholder Instagram links with the real business Instagram URL.
2. Replace the `ADP` logo mark with the real logo image if desired.
3. Add a real email address in `script.js` by changing `mailto:?subject=` to `mailto:your@email.com?subject=`.

## How to make it live online with Netlify
1. Go to netlify.com and create a free account.
2. Click **Add new site** > **Deploy manually**.
3. Drag the entire project folder into Netlify.
4. Netlify gives you a live link instantly.
5. You can later connect a custom domain.

## How to make it live online with GitHub Pages
1. Create a GitHub account.
2. Create a new repository named `arizona-deluxe-painting`.
3. Upload these project files.
4. Go to **Settings** > **Pages**.
5. Under **Branch**, choose `main`, then save.
6. GitHub gives you a live site link.
